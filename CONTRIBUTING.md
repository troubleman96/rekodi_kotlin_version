# Contributing to Rekodi

## Setup

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or device (API 26+)

## Code Style

- Kotlin code follows the official Kotlin style guide
- Use `ktlint` for formatting
- No comments in production code unless essential

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run `./gradlew ktlintCheck` and `./gradlew test`
5. Submit a PR
